==========
cv2routine
==========
Welcome to cv2routine, a simple python package to help you with your computer vision projects.
It is a simple wrapper around OpenCV that allows you to easily create a routine for your computer vision project.

    This project is used for learning, so do not use it in serious projects, as it is impossible to guarantee the performance of this package
